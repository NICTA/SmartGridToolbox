Network Tutorial

In this tutorial, we will explore the basics of how to set up and solve models of electricity networks. We are not performing any simulation, and as such we link only to SgtCore, not SgtSim.

The tutorial can be found in the tutorials/IntroductionToSgt subdirectory of the SmartGridToolbox directory. Copy this directory into a convenient work directory. cd into the directory, run autogen.sh (if needed) and configure. Now make to build the two executables, network and network_yaml.

The code is self documenting. Briefly, network manually creates and solves a network power flow problem. Normally, one wouldn't explicitly code all the network components, as is done in this first example. Instead, objects would be loaded in from a YAML configuration file.

network_yaml loads the same problem as network in from the configuration file network_yaml.yaml, and again solves the network, achieving the same solution.
