Detailed Description

Basic battery class, deriving from DCPowerSourceBase.

Units: SmartGridToolbox system of units uses MW for power. For the Battery class, we shall therefore measure SOC in units of MWh.

Sign convention: like other components, a positive power means injection into the grid. Therefore, charging is positive and discharging is negative.
