Detailed Description

template<typename T>
class Sgt::ComponentCollection< T >

Utility container type combining aspects of map and vector.

May be iterated over or indexed by a string key or an integer index. ComponentPtr and ConstComponentPtr are a kind of smart pointer to elements in the container. They are effectively equivalent to storing an iterator to a map, and as such, are stable under changes to the mapped type. Thus we can obtain a ComponentPtr, and later replace the value stored at the given key with another value, and the ComponentPtr will reference the newly replaced value.
