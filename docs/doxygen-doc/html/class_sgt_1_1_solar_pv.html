◆ setTRef()

void Sgt::SolarPv::setTRef ( double  TRef)
inline

Set the reference temperature.

Parameters
TRefTemperature in K of cells under reference conditions, should be given in datasheet. The reference temperature is usually taken to be 25 C, although the nominal operating cell temperature (NOCT) is usually higher than this. See http://www.pveducation.org/pvcdrom/modules/nominal-operating-cell-temperature.

Parameters
