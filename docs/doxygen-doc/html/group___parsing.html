Parsing classes, functions etc.

SmartGridToolbox uses the YAML language for human readable configuration files. It provides a Parser class to parse YAML into any object, via the Parser<T>::parse(...) command. The ability to parse specific YAML constructs is handled in a modular and extensible way, using derived classes of the ParserPlugin class.
