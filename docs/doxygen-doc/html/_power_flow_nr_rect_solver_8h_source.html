22 // Terminology:
23 // "Bus" and "Branch" refer to n-phase objects i.e. they can contain several phases.
24 // "Node" and "Link" refer to individual bus conductors and single phase lines.
25 // A three phase network involving buses and branches can always be decomposed into a single phase network
26 // involving nodes and links. Thus use of buses and branches is simply a convenience that lumps together nodes and
27 // links.
