Properties Tutorial

In this tutorial, we will see how SmartGridToolbox Properties work.

The tutorial can be found in the tutorials/Properties subdirectory of the SmartGridToolbox directory; please see the file properties.cc. Copy this directory into a convenient work directory. cd into the directory, run autogen.sh (if needed) and configure. Now make to build executable, properties. Run it by typing "properties".

The code is self documenting. See also the explanation here.
