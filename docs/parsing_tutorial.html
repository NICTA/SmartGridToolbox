Parsing Tutorial

In this tutorial, we will explore the SmartGridToolbox mechanism for parsing YAML configuration files. SmartGridToolbox uses YAML as the primary language by which to store configuration data such as network data or simulation setups. A Parser object is used to parse configuration files. Parser functionality is extended by subclassing the ParserPlugin base class.

The tutorial can be found in the tutorials/parsing subdirectory of the SmartGridToolbox directory. Copy this directory into a convenient work directory. cd into the directory, run ./autogen.sh (if needed) and ./configure. Now make to build the executable, parsing.

The code is self documenting. You should open windows containing the source code, parsing.cc, and the YAML configuration files, network.yaml and int.yaml. Next, run ./parsing, and examine the output with reference to the code.
