Simulation Tutorial

In this tutorial, we will explore the basics of how to set up and run a simulation.

The tutorial can be found in the tutorials/Simulation subdirectory of the SmartGridToolbox directory; please see the files simulation.cc, simulation.yaml and simuation.m. Copy this directory into a convenient work directory. cd into the directory, run autogen.sh (if needed) and configure. Now make to build executable, simulation.

The code is self documenting. Briefly, simulation.cc first parses the file simulation.yaml into a Simulation object. The YAML file contains instructions to parse in a network from a Matpower file, and then to add certain objects such as solar PV collectors and inverters. After parsing the input file into the Simulation, simulation.cc demonstrates how the simulation can be manually modified in-code. Before running the simulation, Simulation::init() is called to make sure the simulation and all its component objects are in a valid starting point. A loop is then set up to advance the simulation step by step, extracting and printing information at each step.
